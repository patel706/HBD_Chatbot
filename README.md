# Honeybee Digital — CityHangAround Chatbot App

A premium, full-stack chatbot and business discovery platform. It features a React-based frontend dashboard, a FastAPI python backend API, automated SQLite search matching, and OpenRouter-powered LLM conversational assistance.

## Key Features

- **Conversational Search:** Ask the chatbot for local businesses in natural language (e.g., "Best Pizza" or "Ayurvedic Center in Delhi"). The assistant translates queries, queries the local SQLite database, and offers online search fallbacks.
- **Smart Suggestions:** Auto-predictive suggestions generated by OpenRouter dynamically assist input as users type.
- **Business Dashboard Onboarding:** Business owners can register their business, manage product listings, add deals, and update business profiles via automated wizards inside the chat.
- **OTP Verification:** Verification codes sent via email/SMTP or phone to securely claim listings.
- **Live Health Status:** Built-in connection indicator in the chat header, showing a live "Connected / Offline" status checker.

## Environment Variables Setup

Ensure you create or update the file `backend/.env` containing the following configuration:

```env
PORT=5000                                      # Backend port
FRONTEND_ORIGIN=http://localhost:5173          # Allowed CORS origin
DATABASE_URL=backend/google_map_data.db         # SQLite db file path

# OpenRouter API Keys (LLM Integration, comma-separated keys)
OPEN_ROUTER_API_KEYS="sk-or-v1-..." 

# Email SMTP Verification (Optional, for email OTP)
SMTP_EMAIL="your_email@gmail.com"
SMTP_PASSWORD="your_app_password"
```

## Local Installation & Running

### 1. Backend Server Setup
From the project root:
```powershell
cd backend

# Create a virtual environment inside backend directory
python -m venv .venv

# Activate the virtual environment
# Windows:
.\.venv\Scripts\Activate.ps1
# macOS/Linux:
source .venv/bin/activate

# Install requirements
pip install -r requirements.txt
```

To run the backend server, simply double-click `START_SERVER.bat` inside the `backend` folder, or run:
```powershell
.\.venv\Scripts\python.exe -m uvicorn api:app --host 127.0.0.1 --port 5000 --reload
```

> [!NOTE]
> `START_SERVER.bat` will check for the SQLite database `google_map_data.db` automatically at startup. If it is missing, it will run the migration script `migrate_data.py` to compile it from the source CSV.

### 2. Frontend Development Server Setup
From the project root:
```bash
# Install dependencies
npm install

# Run the frontend dev server
npm run dev
```
The client dashboard will launch at `http://localhost:5173/`.

### 3. Quick Windows Launch (Both Servers Concurrently)
To run both servers instantly at the same time, execute:
```cmd
START_FULL_APP.bat
```
This script runs the React dev server and calls `backend/START_SERVER.bat` in a new window.
